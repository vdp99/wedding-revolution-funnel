# CLAUDE.md

This file provides guidance to Claude Code when working in this directory.

## Workflow di sviluppo

Usa Puppeteer per scattare screenshot durante la costruzione del sito. Dopo ogni fase importante, avvia il server locale, scatta screenshot delle sezioni modificate e fai almeno due passate di revisione e correzione prima di mostrarmi il risultato.

## Tecniche di design

Consultare sempre `reference/tecniche-design.md` per stack tecnologico (Tailwind, ShadCN), animazioni (GSAP, Framer Motion), grafica 3D, tipografia, background, e regole anti-AI slop.

## Landing Page — Style Guide
